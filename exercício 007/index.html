Principais Formatações

Negrito/Destaque

Neste parágrafo temos um termo em negrito usando a tag <b> não semantico.

Nesta frase temos um termo em negrito com a tag <strong>.

Itálico/Ênfase

Neste parágrafo temos uma frase em Itálico não Semântico usando <i>.

Neste parágrafo temos um termo em Itálico com valor Semântico usando <em>.

Texto Marcado

Podemos criar também um texto marcado com a Tag <MARK> com a Formatação de cores das CSS's usando um "Seletor" dentro do "Cabeçalho".

E neste outro parágrafo temos outro texto marcado pela mesma tag e sob a mesma formatação do texto anterior.

Texto Grande e Pequeno

Neste parágrafo foram criadas duas frases, uma grande, com a tag <big>(já obsoleta no HTML-5), e uma pequena com a tag <small>.

Texto Deletado

Podemos marcar um texto como "excluído"com a tag <dell> para indicar que ele deve ser lido mas não deve ser considerado.

Texto Inserido

Podemos marcar um texto "inserido" com a tag <ins> para dar uma ênfase e indicar que ele foi adicionado depois.

Texto Sobre-escrito

Usado para criar coisa como x20+3 (tag <sup>).

Texto Sub-Escrito

Usado para criar coisas como Log10 2 (tag <sub>).
